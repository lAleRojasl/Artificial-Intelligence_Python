1) Asegurarse que tiene instalado python 2.7 en la maquina (estoy usando ubuntu 16.04)


2) Instalar estos paquetes:
	
	sudo apt-get install python-numpy
	
	sudo apt-get install python-matplotlib


3) Abrir una terminal en la carpeta donde estan los archivos.


4) Ejecutar el programa con: 
	
	python plotNumber.py

4.1) Puede que salga un warning la primera vez que lo corra diciendo algo de
construyendo cache y puede que dure un rato pero solo es una vez.
