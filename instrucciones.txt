1) Instalar python 2.7 en la maquina (preferiblemente Ubuntu 16.04)


2) Instalar los siguientes paquetes:
	
	sudo apt-get install python-numpy
	
	sudo apt-get install python-matplotlib


3) Abrir una terminal en la carpeta donde se ubican los archivos.


4) Ejecutar el programa con: 
	
	python plotNumber.py

4.1) La primera vez se mostrará un warning mencionando construcción de cache y puede tardar unos momentos, solo ocurre una vez.
